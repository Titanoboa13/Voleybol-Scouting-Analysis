# 🏐 Volleyball Scouting & Decision Support System

This project is an interactive decision-support tool designed for professional volleyball coaches and scouting teams. It analyzes athlete performance data to identify **athletic anomalies**—players who demonstrate exceptional explosive power relative to their height.

## 🚀 Overview
In high-level volleyball, recruitment and scouting often rely heavily on traditional height standards. This project challenges that norm by providing a data-driven framework to:
- Discover "hidden gems" with elite jumping capacity who might be overlooked.
- Establish mathematical benchmarks for physical performance across different positions.
- Visualize the correlation between physical metrics and on-court potential.

## 📊 Technical Deep Dive
- **Data Engineering:** Used **Python (Pandas)** to clean, filter, and process a dataset of 10,000+ professional players, ensuring data integrity by removing duplicates and handling missing values.
- **Custom Metric (Jump Power):** Engineered a specific performance indicator—**Jump Power**—which measures the delta between spike reach and standing height to quantify raw explosiveness.
- **Interactive Dashboard:** Built a real-time web interface using **Dash** and **Plotly**, allowing scouts to apply dynamic filters (e.g., Min Spike Reach, Max Height) and get instant visual feedback.
